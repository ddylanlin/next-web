依照長程徒步的科學根據，背包重量盡量不超過自身體重的10%為標準。但我從東歐開始擔國際志工與沿路拜訪朋友的旅程中，一路上所需的裝備行頭還有我的電腦零零總總加起來背包的重量大概自己體重的18-20%了。能在路上找到比我還重的包包幾乎是不太可能，路上的朋友看到我駝著厚重的家當總是同情又揶揄的笑我：「LIN carries his home」

<br/>

許多人建議我寄行李到下個城鎮，每天都輕裝上路讓雙腳好好休息復原，但我總是回絕。

 <br/>

我任性的認為背包塞滿我流浪的家當，他是我的同伴、是我的家人、是我的一部份。我受惠於他取之於他，相對的我得為他負責。就算痛苦來自超額的負重，但他在我肩上腳上給我的疼痛苦難都是我得在這條路上承受經歷的，沒有這些重量與苦痛我的旅途就不完整。

 <br/>

在長程徒步之前雖然沒有做過訓練，但平常有持續運動的習慣，雙腿的肌力整天跋涉下來還不到疲累的程度，甚至覺得每天都走不夠想繼續走，但是因為超出身體許多的額外負重，加上長時間的運動及壓迫，使我的關節異常吃不消，關節疲憊拉扯到周圍肌腱，於是我雙腳腳踝附近的肌腱在我每天不間斷地操勞下就這樣一直維持在紅腫發炎疼痛的狀態下沒有任何喘息的機會。

 

 <br/>

背包像門人生的學問。揹上背包旅行一段時間後，你會清楚知道它有多重給你多少負擔。

 <br/>

會開始捨去不必要的東西、不再買不需要的東西；丟東西的時候百般猶豫不捨、買東西的時候更是萬般的猶豫不決，得讓自己背包裡留下的每樣東西都達到它最大的效用或價值，否則都是多餘的可以被替換。而到了最後，你會發現那些背包裡留下來的每一樣東西才是自己真正需要的。

 <br/>

背包的重量，你得為它負責。裝的東西越多你越方便越舒適，但同時你就得為它承受更重的負擔。會開始去思考什麼是自己需要，而什麼是想要，多一份慾望的同時也多一份相應的代價。學習整理背包的時候還真像學習人生慾望取捨的過程！尤其是身在「被強迫消費」時代下的我們，那些我們覺得想要且必須擁有的事物，其實很多時候大部份都不是我們真正需要的，而是因為「廣告」讓這個社會，這個環境決定我們需要什麼。如果能在日常購物時想到「旅行背包」的概念，意識到想要與需要，這過程中即會減少大量的浪費與「越消費越空虛」的心理狀態，且這過程中更能清楚的認識到自己的真正所求。

 

 <br/>

「我們的慾望是模仿他人慾望。」物質是，價值觀更是。
